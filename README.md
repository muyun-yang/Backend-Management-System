# Backend-Management-System

Independently developed a prototype for an AI-powered Q&amp;A interface for public services by integrating with Alibaba’s Tongyi large language model API to provide intelligent, automated solutions.

## 🔧 Tech Stack

- **Framework**: Java, Spring Boot, Spring Cloud
- **Service Discovery**: Nacos
- **Cache**: Redis
- **Containerization**: Docker
- **LLM Integration**: Alibaba Tongyi (Qwen) API
- **Database**: MySQL, MyBatis

## 📌 Key Features

- **Data Visualization Dashboards**: Real-time analytics and reporting
- **Intelligent Q&A Interface**: AI-powered public service assistant powered by Tongyi LLM
- **Microservices Architecture**: Scalable and maintainable service decomposition
